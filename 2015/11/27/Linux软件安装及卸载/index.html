Linux软件安装及卸载

  • Linux

deb包

dpkg命令

  • 安装
    dpkg –i <.deb 文件名>
    如：dpkg –i software-1.2.3-1.deb
  • 卸载
    dpkg –r <软件包名>
    如：dpkg –r software

apt-get命令

  • 安装
    apt-get install <软件包名>
    如：apt-get install software
  • 卸载
    apt-get remove <软件包名>
    如：apt-get remove software

对比：dpkg只能安装已经下载到本地机器上的deb包. apt-get能在线下载并安装deb包,能更新系统,且还能自动处理包与包之间的依赖问题,这个是dpkg工具所不具备的。
更多详细参数可使用dpkg —help和apt-get —help查看。

rpm包

rpm命令

  • 安装
    rpm –ivh <.rpm 文件名>
    如：rpm -ivh software-1.2.3-1.i386.rpm
  • 卸载
    rpm –e <软件包名>
    如：rpm –e software

yum命令

  • 安装
    yum -y install <软件包名>
    如：yum -y install software
  • 卸载
    yum -y remove <软件包名>
    如：yum -y remove software

对比：rpm 只能安装已经下载到本地机器上的rpm 包. yum能在线下载并安装rpm包,能更新系统,且还能自动处理包与包之间的依赖问题,这个是rpm 工具所不具备的。

bin文件

  • 安装
    1. chmod +x soft.bin #增加soft.bin执行权
    2. ./soft.bin

tar.gz源代码包

解压后安装前先阅读附带的INSTALL文件、README文件。

  • 安装
    1. tar –xvzf soft.tar.gz //一般会生成一个soft目录
    2. cd soft
    3. ./configure
    4. make
    5. make install
  • 卸载
    Kinstall和Kife是tar包安装、卸载的黄金搭档。

tar.bz2源代码包

  • 安装
    1. tar -xjvf soft.tar.bz2 //一般会生成一个soft目录
    2. cd soft
    3. ./configure
    4. make
    5. make install
  • 卸载
    Kinstall和Kife是tar包安装、卸载的黄金搭档。

static与const区别

  • C++
  • const
  • static

区别在哪

偶然间，遇到【static与const都有哪里区别】这样一个问题！由于以前只是粗略了解，既然遇到了问题，那么就来个彻底吧！~ 例子中把const static（static const）也加了进来！
经过查阅资料和测试，得到了下列这样一个结果：

例子检验

实践才是检验结果的最好的方法！接下来看看例子。。。

（注意：以下代码均在 win8.1 x64 Dev-C++ 5.4.0 环境下测试所得结果）

先看作为类成员变量时&初始化位置的例子

    #include <iostream>
using namespace std;
class A{
    public:
        const int a;        //a在此初始化编译器会抛出警告，但可运行 
        static int b;
        const static int c=2;        //c可以在此初始化 
        int d;
        A():a(0){        // a在初始化成员列表，b和c不能使用成员列表初始化 
        }

        A(int n):a(n){
        } 

}; 
int A::b=1;        //b只能这样初始化 
//const int A::c=2;        //c也可以这样初始化 

int main()
{
    cout<<endl<<"----- testA -----"<<endl;
    A testA;
    cout<<"a="<<testA.a<<endl;
    //cout<<"a="<<A::a<<endl;        //错误！a为非静态成员 
    cout<<"b="<<A::b<<endl;        //与类相关 
    cout<<"c="<<A::c<<endl;


    cout<<endl<<"----- testA2 -----"<<endl;
    A testA2(3);
    cout<<"a="<<testA2.a<<endl;        //虽然a为const，但对整个类而言是可变的 


    return 0;
}

作为类成员函数的情况

修改一下上面的例子

#include <iostream>
using namespace std;
class A{
    public:
        const int a;         
        static int b;
        const static int c=2;        
        int d;
        A():a(0){         
        }

        static void staticPrint(){
            cout<<"----- staticPrint() -----"<<endl; 
            //cout<<"a="<<a<<endl;        //错误！不能访问non-static成员变量 
            cout<<"b="<<b<<endl;
            cout<<"c="<<c<<endl;
            //print();        //错误！不能访问non-static成员变量 
            cout<<"----- end staticPrint() -----"<<endl; 
        }

        void print(){
            cout<<"----- print() -----"<<endl; 
            cout<<"a="<<a<<endl;
            cout<<"b="<<b<<endl;
            cout<<"c="<<c<<endl;
            cout<<"----- end print() -----"<<endl; 
        } 

        void constFun() const{
            //d=5;        //错误！不能修改对象内容 
            //print();         //错误！不能访问non-const成员函数 
            A::staticPrint();         //可以访问static成员函数 
        } 
}; 
int A::b=1;        
//const int A::c=2;        

int main()
{
    A::staticPrint();         //调用静态成员函数 

    cout<<endl<<"----- testA -----"<<endl;
    A testA;
    testA.print();        // 调用non-static成员函数 

    return 0;
}

运行结果：

再看一个赋值的例子

int main()
{
    cout<<"----- 赋值 -----"<<endl;
    static int x=3;
    const int y=x;
    static const int z=y;        //均可相互赋值 

    cout<<"x="<<x<<endl;        //x=3
    cout<<"y="<<y<<endl;        //y=3
    cout<<"z="<<z<<endl;        //z=3

    return 0;
}

结果x、y、z均可相互赋值，也就是说const变量和static变量均在运行期初始化。
（以上结果均为个人测试所得，如有错误，欢迎指出交流。）
